Getting Started

1. Prerequisite

CB4J requires a Java 1.5+ runtime.

2. Download CB4J

You can download the latest binary release version of CB4J here.

3. Build CB4J

3.1 Get source code

CB4J uses Git as source control manager and is hosted at Github. To download CB4J source code, clone it's git repository as follows :

git clone https://github.com/benas/cb4j.git

You can also get the source code in a zip file from the home page of the project at Github : https://github.com/benas/cb4j

3.2 Build with Maven

CB4J uses maven as build tool. To build CB4J with maven, use the following command from the root directory of the project :

mvn install

After the build is finished, the cb4j-core-{version}.jar file will be added to your local maven repository in the following directory :

net/benas/cb4j/cb4j-core/{version}/cb4j-core-{version}.jar

4. Use CB4J

To use CB4J, you have to include cb4j-core-{version}.jar in your application's classpath.

If you are using maven, add the following dependency to your project pom :

    <dependency>
        <groupId>net.benas.cb4j</groupId>
        <artifactId>cb4j-core</artifactId>
        <version>${version}</version>
    </dependency>
Info CB4J core jar file is not yet published to any maven repository, you have to build and install it to your local maven repository before adding it as a dependency in your project's pom.
