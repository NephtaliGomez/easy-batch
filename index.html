Overview

What is CB4J?

CSV Batch For Java is a framework that aims at simplifying CSV files batch processing with Java.

CB4J uses a POJO-based development approach which lets you process CSV files with an object view of records.

The framework addresses common tasks such as reading, parsing and validating CSV records and lets you concentrate on your data validation and batch processing business logic.

CB4J is not...

  • just an utility class to parse CSV records.
  • a tool to map CSV records to Java objects.

Features

CB4J is a lightweight framework providing the following features :

  • Easy to learn : CB4J uses a very simple CSV model and API
  • A small memory footprint (No dependencies): only a 30 Ko jar file !
  • Built-in extensible field validators for common requirements
  • Simple and customizable reporting for common statistics including validation errors and processing time
